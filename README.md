# DolceBar Angular

DolceBar Angular is a cafeteria and e-commerce web application built with Angular.

## Features
- Product browsing and category navigation
- Product details pages
- Shopping cart management
- User registration and login
- User profile pages
- Admin features for managing users and products

## Technologies Used
- Angular
- TypeScript
- HTML
- CSS
- Local JSON data

## Run Locally
```bash
npm install
ng serve
